## FOR CREATING A NEW REPOSITORY IN GIT
1. Create a new project folder. You can create this folder using VSCode or through your file system.
2. Open a terminal in the project folder:
3. Run git init in the terminal:
4. git init ##used to initialize a new Git repository. When you run this command in a directory, Git sets up the necessary data structures and files to start version controlling your project in that directory. It essentially turns a regular directory into a Git repository.
5. git add .
6. git commit -m "Initial Commit"
7. create the repo in github and get the url
8. git config --global http.sslbackend schannel 
9. git remote add origin <your-github-repository-url>
10. git push -u origin master
